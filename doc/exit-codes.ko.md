종료 코드
=========

`dojang` 명령은 오류로 인해 종료될 때 상황에 따라 아래와 같은 종료 코드를
반환합니다.

 -  -1: 예기치 못한 오류.
 -  1: 잘못된 명령 옵션 또는 인자.
 -  2: 파일 쓰기 실패.
 -  3: 파일이 없음.
 -  4: 외부 프로그램이 오류로 종료됨 (0이 아닌 종료 코드).
 -  9: 해당 환경(플랫폼)에서 지원하지 않음.
 -  10: 선언 파일이 없거나 저장소가 초기화되지 않음.
 -  11: 잘못된 형식이나 권한 또는 기타 파일 시스템 문제로 선언 파일을 읽거나
    갱신할 수 없음.
 -  12: 선언 파일이 이미 있어서 새로 만들 수 없음.
 -  13: 머신 상태를 읽거나 갱신할 수 없거나, 식별자가 없거나,
    상태가 손상·미지원·모호한 상태이거나, 다른 머신의 상태임.
 -  20: 환경 파일이 없음.
 -  21: 환경 파일을 잘못된 형식이나 권한 등의 문제로 읽을 수 없음.
 -  22: 비대화형 머신 등록에 필요한 머신 정보가 빠짐.
 -  30: 저장소와 적용 대상 사이에 충돌하는 변경이 각자 있어서 변경을 적용할 수
    없음.
 -  31: 저장소 내의 파일은 대상 파일이 될 수 없음.
 -  32: 경로가 지정되지 않은 파일이라 작업을 수행할 수 없음.
 -  33: 무시된 파일이라 작업을 수행할 수 없음.
 -  34: 대상 데이터가 삭제되거나 수정된 대상의 로컬 판정 근거가 사라질 수 있어서
    작업을 취소함.
 -  35: 여러 경로 규칙이 같은 대상에 대응하여 선택이 모호함.
 -  36: 사용자가 작업을 취소함.
 -  37: 수명 주기 선택이 비어 있거나 알 수 없거나 아직 활성 상태임.
 -  38: 라우트 설정이 각 목적지에 정확히 하나의 소유 라우트를 부여할 수
    없거나, 라우트의 원본과 목적지가 서로 겹침.
 -  40: 훅이 0이 아닌 종료 코드로 실패함.
